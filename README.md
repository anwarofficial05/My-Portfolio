## ✨ Features

- **Typing animation** — hero role cycles through Data Analyst, ML Enthusiast, Frontend Developer, and more
- **Scroll-reveal animations** — sections and cards animate in as you scroll
- **Animated skill bars** — progress bars trigger on scroll into view
- **Sticky navigation** — blur backdrop nav fixed at the top
- **Fully responsive** — works on mobile, tablet, and desktop
- **Single file** — all HTML, CSS, and JS in one file, zero build step needed
- **Dark theme** — deep navy background with electric cyan accents
- **No frameworks** — pure vanilla HTML, CSS, and JavaScript

## 🚀 Getting Started

### Run Locally

```bash
# Clone the repository
git clone https://github.com/anwarofficial05/portfolio.git

# Navigate into the folder
cd portfolio

# Open in browser
open index.html
# or just double-click index.html
```

No installs. No build tools. No dependencies. Just open and go.

### Deploy to GitHub Pages

```bash
# 1. Push your code to GitHub
git add .
git commit -m "Initial portfolio commit"
git push origin main

# 2. Go to your repo → Settings → Pages
# 3. Set Source: Deploy from branch → main → / (root)
# 4. Your site will be live at:
#    https://anwarofficial05.github.io/portfolio
```
