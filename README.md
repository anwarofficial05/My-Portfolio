# 📜 Certification Management

This portfolio uses a separate JSON file.

```
certificates.json
```

Adding a certificate only requires:

1. Open Manage Log
2. Add Certificate
3. Download updated JSON
4. Replace the existing JSON file

No HTML editing required.
